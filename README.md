## 📌 Project Overview
Analyzed **1,00,000 real loan records**
from Lending Club to identify
default risk patterns, grade-wise
performance, and borrower behavior.

## 🗂️ Dataset
| Detail | Info |
|--------|------|
| Source | Lending Club — Kaggle |
| Rows | 1,00,000 |
| Columns | 15 |
| Period | 2014 – 2018 |

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| MySQL | 12 Business SQL Queries |
| Python | EDA + Risk Analysis |
| Power BI | 3-Page Dashboard |

## 📊 Key Business Insights
1. Overall default rate: **11.6%**
2. Grade G loans = **36.4% default**
   *(12x higher than Grade A at 3.1%)*
3. Small business = highest default risk
4. 60-month loans = **2x default rate**
5. Higher DTI = Higher default risk
